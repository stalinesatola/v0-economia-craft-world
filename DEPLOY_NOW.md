# 🎯 COMEÇAR AGORA - Deploy em 3 Passos

## PASSO 1: Abra Vercel
https://vercel.com/new

## PASSO 2: Conecte GitHub
- Clique "Import Project from GitHub"
- Procure por: **stalinesatola/v0-economia-craft-world**
- Clique "Import"

## PASSO 3: Adicione Senha
- Na tela "Configure", vá para "Environment Variables"
- Clique "Add New"
- **Name:** `ADMIN_PASSWORD`
- **Value:** Digite uma senha forte (ex: `CraftWorld@2024!Secure`)
- Clique "Deploy"

## ✅ PRONTO!

Aguarde 3-5 minutos e sua app estará online!

### Acessar
```
https://seu-projeto-name.vercel.app
```

### Login
- User: `admin`
- Password: `{a-senha-que-voce-digitou}`

## 🆘 Problemas?

1. **Build Failed?**
   - Verifique `ADMIN_PASSWORD` env var está lá
   - Tente novamente: "Deploy" button no Vercel

2. **Database Error?**
   - Espere mais um pouco (Neon pode levar 1-2 min)
   - Refresque a página

3. **Pode't Login?**
   - Verifique se digitou a senha correta
   - Reinicie o browser (Clear cache)
